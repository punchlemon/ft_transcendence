# Release Notes — v1.0.0 Candidate

日付: 2025-11-29

概要:
- 基本機能（認証、ユーザー管理、トーナメント、対戦ゲーム、チャット）が安定動作することを確認。
- CI (Vitest) によるユニット/統合テストが全てグリーン（バックエンド: 68/68, フロントエンド: 74/74）。
- Docker Compose による backend/frontend の同時起動が可能になり、CI でのビルド検証を追加。

主要変更点:
- 認証: `/auth/register`, `/auth/login`, `/auth/refresh`, `/auth/logout` の実装と MFA (TOTP + バックアップコード) の統合。
- OAuth: PKCE 対応の認可 URL 発行とコールバック処理の実装。
- トーナメント: 作成/一覧 API とフロント統合、マッチ生成ロジックを追加。
- ゲーム: サーバ権威の Pong エンジン（120Hz ループ、AI 1Hz）、`GameRoom` の Canvas UI を実装。
- Chat/Notification: WebSocket ベースの通知/チャット、`NotificationService` を実装。

既知の制約／注意点:
- 本番デプロイ前に OAuth プロバイダのクレデンシャル設定とリダイレクト URL の整合性確認が必要。
- データ移行手順（マイグレーション）は環境によって差異があるため、デプロイ時に確認すること。

次のステップ:
1. リリースノートのレビューと加筆（必要に応じて変更履歴を細分化）。
2. `v1.0.0-candidate` タグをレビュー用に push（必要なら CI を通す）。
3. 本番デプロイ手順の最終確認（DB マイグレーション、環境変数、シークレット管理）。
